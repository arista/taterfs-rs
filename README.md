# taterfs-rs

A Rust command-line utility.

## Status

This project is in early development.

## Prerequisites

- Rust (install via [rustup](https://rustup.rs/))

## Building

```bash
cargo build
```

## Running

```bash
cargo run
```

## Testing

```bash
cargo test
```
